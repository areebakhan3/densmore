Embed This Widget on Any Page

Add just two lines to any HTML page. The chat bubble will appear in the bottom-right corner.

Copy & Paste into your <head> and before </body>

<!-- Step 1: Add CSS in <head> -->
<link rel="stylesheet" href="http://localhost:3000/widget/iife.css">

<!-- Step 2 (optional): Custom config -->
<script>
  window.DensmoreChatConfig = {
    apiBase: 'http://localhost:3000',  // your server URL
    title:   'Insurance Assistant',
    subtitle:'Ask me anything',
  };
</script>

<!-- Step 3: Add JS before </body> -->
<script src="http://localhost:3000/widget/iife.js"></script>
